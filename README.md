# MV Studio — 紹介サイト

声で相談してつくる、AI映像ディレクター「MV Studio」のプロモーションサイト。
夜のオーロラから夏のプールへと移ろう 3D スクロール。日本語／繁體中文（台灣）対応。

## 構成
- `index.html` — マークアップ
- `css/styles.css` — スタイル
- `js/main.js` — WebGL背景・スクロール演出・言語切替

## ローカルで見る
```
python3 -m http.server 8000
# http://localhost:8000
```
